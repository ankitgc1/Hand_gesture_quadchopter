# Hand_gesture_quadchopter
A Hand gesture control quadchopter. 

### Required things :-
##### For Transmitter part:-
1. Arduino mini/micro
2. NRF module 
3. Potensiometer
4. MPU6050(IMU sensor)
5. Push button
6. 9V battery
7. Glove
8. 2 LEDs

##### For Receiver part:-
1. Drone frame
2. CC3D flight controller
3. 4 BLDC motors
4. 4 ESCs
5. 4 Propellers
6. NRF module
7. Arduino mini/micro
8. Ultrasonic sensor
9. Lipo battery
10. Power distributer
